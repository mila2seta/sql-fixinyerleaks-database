# Fixinyerleaks SQL Database Project

Microsoft SQL Server database project developed for Database Design and SQL I.

## Project Overview

This project uses a relational database for a residential plumbing company. The database tracks customers, service jobs, workers, worker hours, materials, vendors, job statuses, and service types.

## Features

- Designed and analyzed a relational database structure
- Worked with primary keys and foreign keys
- Used an ERD to understand table relationships
- Wrote SQL queries to generate business reports
- Verified query results against expected result sets

## SQL Concepts Practiced

- Single-table queries
- INNER JOINs
- LEFT OUTER JOINs
- Aggregate functions: COUNT, SUM, AVG, MIN, MAX
- GROUP BY and HAVING
- WHERE filtering
- ORDER BY sorting
- Subqueries
- LIKE filtering

## Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- SQL
